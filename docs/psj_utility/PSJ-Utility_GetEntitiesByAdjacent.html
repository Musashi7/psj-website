    GetEntitiesByAdjacent¶

    Description¶

    Get list of entity object by adjacency

    Inputs¶

    1. DItem enum type (JPT.DItemType)

    2. id entity (int)

    3. stop angle (int)

    Return Code¶

    list of DItem object (ItemVector)

    Sample Code¶

    JPT.GetEntitiesByAdjacent(JPT.DItemType.BODY, 1, 30)
