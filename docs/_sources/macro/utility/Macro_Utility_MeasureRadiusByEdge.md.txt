MeasureRadiusByEdge
-------------------

## Description

Measure the angle created by edge.

## Syntax

MeasureRadiusByEdge(cursor edge,Integer N)

## Inputs

1.  cursor

    edge cursor(11:\*,\*=edge id)

2.  Integer N

    specify the number of decimal places (0\<=N\<=30)

## Return Code

Angle value

Succeeded

FAILED

Failed

## Sample Code

MeasureRadiusByEdge(5:11,6)
