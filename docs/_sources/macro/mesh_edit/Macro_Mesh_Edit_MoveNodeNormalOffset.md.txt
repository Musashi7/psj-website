MoveNodeNormalOffset
--------------------

#### Description

Move node(s) in Normal Direction of plane

#### Syntax

MoveNodeNormalOffset(double posMagnitude, int\[\] iNodeKey)

#### Inputs

1.  Double

    Position magnitude

2.  Int\[\]

    Node key cursor(\[Node ID\])

#### Return Code

1

Succeeded

0

Failed

#### Sample Code

MoveNodeNormalOffset(0.002, \[1432\])
