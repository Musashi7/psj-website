CreateNode
----------

#### Description

Create nodes

#### Syntax

CreateNode(double\[3\] taNodeCoord, int\[\] taNodeKey)

#### Inputs

1.  Double\[\]

    Coordinate point(\[xi, yi, zi\])

2.  Int\[\]

    Input Node key (\[Node ID\])

#### Return Code

1

Succeeded

0

Failed

#### Sample Code

CreateNode(\[\[0, 0.005, 0.012\],\[0, 0.005, 0.020\]\], \[1018,1019\])
