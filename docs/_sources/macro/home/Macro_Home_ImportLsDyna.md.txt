ImportLsDyna
------------

## Description

Import Ls-Dyna file

## Syntax

ImportLsDyna(string\[\] m_vecPath,double m_faceAngle,double m_edgeAngle)

## Inputs

1.  String\[\]

    Multiple files Path

2.  Double

    Face angle

3.  Double

    Edge angle

## Return Code

1

Succeeded

0

Failed

## Sample Code

ImportLsDyna(\["D:/ImportFile/LsDyna_Test.k"\], 1.0472, 1.0472)
