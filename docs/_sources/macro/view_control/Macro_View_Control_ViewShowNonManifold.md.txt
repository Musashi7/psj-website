ViewShowNonManifold
-------------------

#### Description

Turn ON/OFF Show NonManifold

#### Syntax

ViewShowNonManifold(bool flag)

#### Inputs

1.  bool

    Turn on/off display switch 0=On 1=OFF

#### Return Code

Nothing

#### Sample Code

ViewShowNonManifold(0)
