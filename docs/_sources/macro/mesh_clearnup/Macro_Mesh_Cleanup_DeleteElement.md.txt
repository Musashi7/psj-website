DeleteElement
-------------

#### Description

Delete Element

#### Syntax

DeleteElement(cursor\[\] elemList, bool bKeepShareElem)

#### Inputs

1.  Cursor\[\]

    Target element cursor(\[11:Element ID\])

2.  Bool

    N/A: Keep Share Element bool flag True = 1, False = 0

#### Return Code

1

Succeeded

0

Failed

#### Sample Code

DeleteElement(\[11:1112\], 0)
