MC_ManualCleanup_2TrisToQuad
----------------------------

#### Description

Merge two Tri element into one Quad element

#### Syntax

MC_ManualCleanup_2TrisToQuad(Cursor\[\] elements)

#### Inputs

1.  Cursor\[\]

    Array of only 2 Tri elements

#### Return Code

1

Succeeded

0

Failed

#### Sample Code

MC_ManualCleanup_2TrisToQuad(\[11:1068,11:1066\])
