# DeleteSubAssembly

## Description

Delete an indicated sub assembly.

## Syntax

DeleteSubAssembly(DItem subAssembly)

## Inputs

1. DItem

 Target sub assembly.

## Return Code

*Python Output*

- No return value

## Sample Code

*Input:*

```python
SubAsm=JPT.FindSubAssemblyByID(1)
JPT.DeleteSubAssembly(SubAsm)
```
