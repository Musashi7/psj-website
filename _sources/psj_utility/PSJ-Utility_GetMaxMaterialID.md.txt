# GetMaxMaterialID

## Description

Get maximum id of material inside a document.

## Syntax

GetMaxMaterialID()

## Inputs

1. None

No input value

## Return Code

*Python Output*

- int

Material ID

## Sample Code

*Input:*

```python
id = JPT.GetMaxMaterialID()
print(str(id))
```

*Output:*

\> `3`
