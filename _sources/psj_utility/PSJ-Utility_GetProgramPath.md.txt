# GetProgramPath

## Description

Get application installation directory

## Syntax

GetProgramPath()

## Inputs

1. None

No input value

## Return Code

*Python Output*

- String

Program Path

## Sample Code

*Input:*

```python
path=JPT.GetProgramPath()
print(str(path))
```

*Output:*

\> `C:\Program Files\TechnoStar\Jupiter-Pre_4.1\`
