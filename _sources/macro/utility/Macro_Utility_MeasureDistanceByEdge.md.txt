MeasureDistanceByEdge
---------------------

## Description

Measure the edge length

## Syntax

MeasureDistanceByEdge(cursor edge,Integer N)

## Inputs

1.  cursor

    edge cursor(5:\*,\*=edge id)

2.  Integer N

    specify the number of decimal places (0\<=N\<=30)

## Return Code

Edge Length

Succeeded

FAILED

Failed

## Sample Code

MeasureDistanceByEdge(5:18,6)
