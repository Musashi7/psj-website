EnableLicenseFeature
--------------------

## Description

Turn ON / OFF indicated license.

## Syntax

EnableLicenseFeature(String Lisense, int ON/OFF)

## Inputs

1.  String

    Lisense feature shown in Home \> Preference \> License.

2.  int

    1: Turn ON the license feature.  
    0: Turn OFF the license feature.

## Return Code

None

## Sample Code

EnableLicenseFeature("JPT_ANNAS", 0)
