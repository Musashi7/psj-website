CaptureToImageFile
------------------

## Description

Capture to Image File.

## Syntax

CaptureToImageFile(String FilePath)

## Inputs

1.  String

    Image file path(extension = \*.bmp or \*.jpg or \*.gif or \*.png)

## Return Code

TRUE

Succeeded

FALSE

Failed

## Sample Code

CaptureToImageFile("C:\\Test.jpg")
