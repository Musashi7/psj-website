ImportAdx
---------

## Description

Import adx files

## Syntax

ImportAdx(string\[\] vecPath, double faceAngle, double edgeAngle)

## Inputs

1.  String\[\]

    Path directory

2.  Double

    Face angle

3.  Double

    Edge angle

## Return Code

1

Succeeded

0

Failed

## Sample Code

ImportAdx(\["D:/Test.adx"\], 1.0472, 1.0472)
