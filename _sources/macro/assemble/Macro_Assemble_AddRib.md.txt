AddRib
------

## Description

Add Rib

## Syntax

AddRib(Cursor part, Cursor\[\] face_list, Cursor\[\] node_list, int width, int depth)

## Inputs

1.  Cursor

    Part

2.  Cursor\[\]

    Face List

3.  Cursor\[\]

    Node list

4.  int

    Rib width

5.  int

    Rib depth

## Return Code

1

Succeeded

0

Failed

## Sample Code

AddRib(3:10, \[6:260, 6:26\], \[\], 10, 0)
