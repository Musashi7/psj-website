HexBoxMesh
----------

#### Description

Box hex mesh creator for parts

#### Syntax

HexBoxMesh(int\[\] taBodyKey, double dMeshSize, string sMaterialName)

#### Inputs

1.  int\[\]

    Body ID

2.  Double

    Box mesh size

3.  String

    Material Name

#### Return Code

1

Succeeded

0

Failed

#### Sample Code

HexBoxMesh(\[1\], 0.002, "Magnesium_Alloy")
