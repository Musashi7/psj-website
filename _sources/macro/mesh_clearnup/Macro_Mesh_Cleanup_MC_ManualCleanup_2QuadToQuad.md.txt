MC_ManualCleanup_2QuadToQuad
----------------------------

#### Description

Merge two Quad elements into one Quad element

#### Syntax

MC_ManualCleanup_2QuadtoQuad(cursor\[\] elemList)

#### Inputs

1.  Cursor\[\]

    Target element cursor(\[11:Element ID\])

#### Return Code

1

Succeeded

0

Failed

#### Sample Code

MC_ManualCleanup_2QuadToQuad(\[11:1047, 11:1045\])
