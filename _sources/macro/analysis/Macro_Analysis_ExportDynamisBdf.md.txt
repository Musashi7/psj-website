ExportDynamisBdf
----------------

## Description

Export Dynamis bdf file

## Syntax

ExportDynamisBdf(String strPath, TCursor job)

## Inputs

1.  String\[\]

    bdf file path

2.  Cursor

    job cursor

## Return Code

1

Succeeded

0

Failed

## Sample Code

ExportDynamisBdf("D:/ImportFile/TS-Solver.bdf", 148:2)
