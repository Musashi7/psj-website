CreateFaceFromFourEdges
-----------------------

## Description

Create Face From Four Edges

## Syntax

CreateFaceFromFourEdges(cursor\[\] vcrEdges)

## Inputs

1.  Cursor\[\]

    Target edge cursor(\[5:Edge ID\])

## Return Code

1

Succeeded

0

Failed

## Sample Code

CreateFaceFromFourEdges(\[5:35, 5:38, 5:37, 5:32\])
