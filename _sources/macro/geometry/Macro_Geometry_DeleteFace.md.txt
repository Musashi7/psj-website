DeleteFace
----------

## Description

Delete Face

## Syntax

DeleteFace(int\[\] taFaceID, bool bKeepSolid)

## Inputs

1.  int\[\]

    Face ID(\[\*\]\*=Face ID)

2.  bool

    Keep Solid flag true=1, false=0

## Return Code

1

Succeeded

0

Failed

## Sample Code

DeleteFace(\[21\], 1)
