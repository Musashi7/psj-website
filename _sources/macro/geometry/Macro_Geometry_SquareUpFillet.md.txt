SquareUpFillet
--------------

## Description

Scale Up Fillet

## Syntax

SquareUpFillet(cursor\[\] vcrFaces)

## Inputs

1.  Cursor\[\]

    Fillet face list cursor (\[6: Face ID\])

## Return Code

1

Succeeded

0

Failed

## Sample Code

SquareUpFillet(\[6:21, 6:26, 6:54\])
