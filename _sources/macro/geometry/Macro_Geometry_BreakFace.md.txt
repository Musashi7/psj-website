BreakFace
---------

## Description

Break selected face

## Syntax

BreakFace(Cursor\[\] FACE Cursor)

## Inputs

1.  Cursor\[\]

    Target face cursor(\[6:Face ID\])

## Return Code

1

Succeeded

0

Failed

## Sample Code

BreakFace(\[6:64, 6:537, 6:531, 6:534\])
