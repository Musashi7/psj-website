ExportMeshSetting
-----------------

## Description

Save local setting

## Syntax

ExportMeshSetting(String fname,int type)

## Inputs

1.  String

    the file name to save local setting

2.  int

    the save mode, in this function the value is fixed to 3

## Return Code

1

Succeeded

0

Failed

## Sample Code

ExportMeshSetting("D:/ImportFile/test.xml", 3)
