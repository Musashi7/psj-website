ExportDynamisBdf
----------------

## Description

Export Dynamis bdf file

## Syntax

ExportDynamisBdf(String strPath, Cursor job)

## Inputs

1.  String

    bdf file path

2.  Cursor

    job cursor

## Return Code

1

Succeeded

0

Failed

## Sample Code

ExportDynamisBdf("C:\\Users\\TECHNO\~1\\AppData\\Local\\Temp\\TechnoStar\\000B07DF-001E-0006-2C00-1600DD002600.dat", 135:1)
