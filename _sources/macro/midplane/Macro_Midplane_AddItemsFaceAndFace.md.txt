AddItemsFaceAndFace
-------------------

## Description

Add Items Edge by projecting Extend Face's Edge at the extended end of Reference Face

## Syntax

AddItemsFaceAndFace(Cursor\[\] Face, int Extend Type)

## Inputs

1.  Cursor \[\]

    Face List

2.  int

    Extend Type \[Extend,Intersect\]

## Return Code

1

Succeeded

0

Failed

## Sample Code

AddItemsEdgeFaceandFace(\[6:62, 6:51\],0)
